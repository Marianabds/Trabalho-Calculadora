# Trabalho-Calculadora

## Requisitos:
 - O aplicativo deve ter botões para números (0-9), para operadores (=,-,x,/,+) e para limpar (DEL,C).
 - O aplicativo deve exibir os elementos selecionados na tela.
 - O aplicativo deve realizar cálculos de adição, subtração, multiplicação e divisão.
 - O aplicativo deve ser capaz de realizar operações com números inteiros e reais.
 - O aplicativo não deve permitir divisões por zero.
